This is a game of tic-tac-toe with a twist. 
After a regular game the players can move 
pieces to any neighboring block. Just like 
the regular game, if at any point a player 
has 3 in a row. The game ends. 

Step 1: Press start game button
Step 2: Press any block to place a piece alternating between players.
Step 3: After 6 moves, you can move your pieces by first clicking the one to be to moved then clicking the destination of this piece.
Step 4: Continue until 3 pieces are in a row.
Step 5: After end game prompt press start game button to restart.

Notes: 
Background can be changed with pieces below.
Score is kept for reference in the top right.
Notes about game status and turn is listed on top of the board.
